# cryptic-cave

## Setup

1. Clone the repository
2. Backend:  
    - `npm install`
    - `cd api`
    - create a `.env` file with the following content:
        ```
        PORT= port number
        MONGO_URL= mongodb uri
        ```
    - `node index.js`
3. Frontend:
    - `cd client`
    - `npm install`
    - `npm run start`
